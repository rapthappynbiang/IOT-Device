# IOT-Device

This Project will simulate iot data, and will also validate the sensor data for alarms, and will notify the person responsible through email.

To Run this Project You will need

- Redis in your localhost

- Set the environmet variables NODEMAILER_EMAIL and NODEMAILER_PASSWORD (email app password)

# 1. API

- /sensor/generateData: method POST
- /sensor/subscribe: method POST
